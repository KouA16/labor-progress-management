## 🎓 システム習得ガイド

### 📚 学習順序（推奨）
1. **まずはクイックリファレンスで概要把握**
2. **役割に応じた詳細ガイドを熟読**
3. **実際のシステムで操作練習**
4. **疑問点は担当部署に確認**

### ⏱️ 習得時間の目安
- **基本操作**：30分程度
- **全機能理解**：1-2時間程度
- **運用習熟**：1週間程度の実践
